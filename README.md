# Description : Klimate

This is an API based Web application ( React + TypeScript + Vite ) to fetch real time weather data of any location or current location.


## Tech Stack :

React + TypeScript + Vite

## Installation :

1. clone this repositiory
2. ```cd Klimate```
3. ```npm install```
4. create your own API Key. Keept it in ```.env``` file by this name ```VITE_OPENWEATHER_API_KEY```
5. ```npm run dev```
